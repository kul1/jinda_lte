# Jinda::Bsb

Convert jquery.mobile to bootstrap AdminBSB theme 
Make sure finish Jinda Installation first.


## Installation

Finish install Jinda
Add this line to your application's Gemfile:

```ruby
gem 'jinda-lte'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jinda-lte
